< 実行方法 >


1. 以下コマンドをコンソールから実行する。
# git clone https://github.com/Sesta/s1513114-sezemi-2015-readable-code.git

2. 必要なファイルが作られている。
3. index.htmlをダブルクリックする。
5. ブラウザのコンソールで showRecipes( 表示したいレシピのID ) を実行する。
4. 実行結果を確認する


< データの用意方法 >

recipe-data.js というファイルで
recipeData という変数が宣言されている。

この変数の中に表示したいレシピ名を文字列の配列で入れれば良い
